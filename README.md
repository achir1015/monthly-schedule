謝謝！這個系統從零開始建起來，工程量不小 😊

**目前系統完成的功能清單：**

| 功能 | 說明 |

|---|---|

| PDF 班表轉入 | 自動辨識民國年月、班別代號，含覆蓋確認 |

| 桌機版班表 | 橫向全員表格，點格子展開班別選單 |

| 手機版班表 | 月曆式，下拉選員工，點格子修改 |

| 桌機/手機切換 | 自動偵測螢幕寬度，也可手動切換 |

| 員工管理 | 新增、編輯、刪除，含證照、入職日期 |

| 班別設定 | 可自訂代號、名稱、文字/背景顏色 |

| XML 備份還原 | 匯出下載、匯入覆蓋，Google Drive 連結 |

| 月份切換 | ‹ › 跨年月切換，自動建立空白月份 |

| 資料持久化 | 瀏覽器 storage 自動儲存 |

| 單檔部署 | `index.html` 直接雙擊或 GitHub Pages 使用 |

GitHub Pages 已更新，但瀏覽器還在讀舊版。
解決方法（擇一）：
方法一：強制重新整理（最快）
Windows：Ctrl + Shift + R
Mac：    Cmd + Shift + R
方法二：無痕視窗開啟

Chrome 右上角選單 → 新增無痕式視窗
輸入網址 achir1015.github.io/monthly-schedule/

方法三：清除快取

Chrome 網址列按 🔒 鎖頭圖示
或按 F12 → Network → 勾選 "Disable cache" → 重新整理
